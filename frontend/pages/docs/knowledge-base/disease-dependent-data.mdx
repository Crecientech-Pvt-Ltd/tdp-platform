# Disease-dependent data

**Disease-dependent** data varies with different diseases, including [LogFC](disease-dependent-data/logfc.mdx) (_Differential expression in Log2 fold change_), [GDA ](disease-dependent-data/gda.mdx)(_Gene Disease Association score_) and [Genetics](disease-dependent-data/genetics.mdx) (_Odd ratio or Beta-values from population studies_).

| Data type | Representation | Experiment types                             | Sources                        |
| --------- | -------------- | -------------------------------------------- | ------------------------------ |
| LogFC     | [-Inf, +Inf]   | RNAseq, Ptoteomics                           | Verge, AD (Mayo, ROSMAP, MSBB) |
| GDA       | [0, 1]         | AI, Meta-scores                              | Open Targets                   |
| Genetics  | [-1, 1]        | GWAS w/ eQTL (OR/Beta), GeneBurden (OR/Beta) | Open Targets Genetics          |
